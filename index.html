browse.js

It is a minimal browser compatibility library that provides APIs for DOM manipulation and AJAX. All APIs have been tested for Chrome 15+, Firefox 3.6+, Opera 15+, IE 6+, Safari 4+, iOS 3+, Android 2+ and other browsers.

Performance Comparison

The Performance Comparison page has tests you can run to compare performance of browse.js with that of jquery, jquery slim, prototype, and mootools.
