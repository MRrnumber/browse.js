Performance Comparison

About the Tests

Each of the tests below creates 100000 elements by appending (using equivalent APIs in each library e.g. append in browse.js, jquery; inject in mootools; and insert in prototype) an element to body.

Since creating and keeping all the elements makes the browser hang typically, the mechanism used is to create an element, measure and remember the time taken, and then delete the newly created element before adding another.

How to run

Press the 'run' button and wait for time taken to be output on its right. Click a test multiple times to see the average time taken, which gives a better and fair idea.
